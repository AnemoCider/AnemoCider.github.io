Developers: Junwen Yu & Yirui Miao

Tech used:

- Unity

    - Event System
- C#
    - Coroutine
    - Interface
- Paradigm
    - State Machine

- Yirui Miao was mainly responsible for:
    - UI design
    - The inventory
    - Enemy design
    - Collectables
    - Bomb implementation
    - Audio system
